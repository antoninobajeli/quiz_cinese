# Quiz Cinese

Applicazione Flutter che legge domande da un file JSON e mostra domande randomiche all'utente.

## Come usare

1. Apri il progetto in Flutter.
2. Esegui `flutter pub get`.
3. Avvia l'app con `flutter run`.

## Struttura

- `lib/main.dart`: logica dell'app e UI.
- `assets/questions.json`: domande del quiz.

## Funzionalità

- Carica domande da JSON.
- Mostra domande randomiche.
- Supporta domande a risposta aperta e a scelta multipla.
- Mostra feedback e punteggio finale.
